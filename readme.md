# MINT COINS - CRYPTO CURRENCY

## Requirements
- django == 5.0.6


## Installation

- `python -m venv venv` (create a virtual environment)
- `venv/scripts/activate` (turn on the virtual environment)
- `python manage.py runserver` (To run the server)
